# vue-test
Aula vue

# Instalar as dependencias
```
npm install
```

# Build do projeto
- Desenvolvimento
```
npm run dev
```
- Produção
```
npm run build
```

# Necessário projeto back-end para funcionar

- Link do repositório 

https://github.com/AlexsonSP/vue-test-api

- Dentro da pasta API execute o comando:
```
npm start
```
